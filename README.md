# Redstone Cowork

You are the CEO. Redstone Cowork simulates a company under you — a virtual team of AI employees that ingests your work streams (Jira, Mattermost, Gmail, Calendar, Claude Code sessions), keeps one clarified & prioritized backlog, discusses decisions like a real team, and relays your calls back to running Claude Code sessions on any machine — from your phone.

Self-hosted, single user per instance, Docker deployment (with bulk multi-employee mode for company servers). Uses your own Claude subscription via the Claude Agent SDK.
